# 🎮 Hangman Game - Python Project

Welcome to my Hangman Game! This is a classic word-guessing game built using Python, focusing on modular programming and logical flow.

## 🚀 Features
- **Modular Structure:** The code is split into three files for better organization (Logic, Art, and Word list).
- **Interactive Gameplay:** Visual hangman stages that update with every wrong guess.
- **Large Word Database:** Integrated a vast list of words to ensure a fresh experience every time.
- **User-Friendly Feedback:** Alerts the user if they repeat a letter or make a wrong guess.

## 🛠️ Technical Skills Demonstrated
- **Loops & Control Flow:** Mastered nested `while` and `for` loops.
- **Lists & Strings:** Handling dynamic string updates and list management for correct/wrong letters.
- **Modular Python:** Importing variables and lists from external `.py` files to keep the main code clean.

## 📂 Project Structure
- `Hange_Man.py`: The main logic and game engine.
- `hangman_art.py`: Contains the ASCII art logo and the 7 stages of the hangman.
- `hangman_words.py`: A massive list of secret words used for the game.

## 🔧 How to Run
1. Ensure you have Python installed.
2. Download the three files into the same folder.
3. Run the main file using:
   ```bash
   python Hange_Man.py
